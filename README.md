# Lista de Exercícios em C# – SENAI

Projeto desenvolvido durante o **Curso Técnico em Desenvolvimento de Sistemas – SENAI**,  
com o objetivo de praticar e consolidar os fundamentos da linguagem **C#** por meio de exercícios progressivos.

---

## Objetivo do Projeto

Esta lista foi criada para fortalecer a base em lógica de programação utilizando C#, evoluindo do nível básico até o uso de coleções.

Os exercícios foram organizados em níveis, cada um focando em um conjunto específico de conceitos.

---

# Nível 1 – Variáveis e Operações Básicas

Neste nível foram praticados:

- Declaração de variáveis
- Tipos primitivos (int, double, string, bool)
- Entrada e saída de dados no console
- Operações matemáticas
- Manipulação básica de valores

Objetivo: desenvolver familiaridade com a sintaxe da linguagem e o funcionamento básico de um programa em C#.

---

# Nível 2 – Estruturas Condicionais

Neste nível foram trabalhados:

- Estruturas `if`, `else if` e `else`
- Operadores relacionais
- Operadores lógicos (`&&`, `||`)
- Validação de dados
- Tomada de decisão baseada em condições

Objetivo: aprender a controlar o fluxo do programa de acordo com diferentes situações.

---

# Nível 3 – Estruturas de Repetição (Loops)

Neste nível foram explorados:

- Loop `for`
- Loop `while`
- Loop `foreach`
- Contadores e acumuladores
- Uso do operador de módulo (`%`)
- Repetição controlada por condição

Objetivo: compreender como automatizar tarefas repetitivas e percorrer dados de forma eficiente.

---

# Nível 4 – Coleções (Arrays e Listas)

Neste nível foram praticados:

- Declaração e uso de arrays
- Manipulação de índices
- Inversão de elementos
- Uso de `List<T>`
- Métodos como `Add()`, `Count`
- Percorrer coleções com `foreach`

Objetivo: aprender a armazenar e manipular múltiplos dados de maneira estruturada.

---

# Tecnologias Utilizadas

- C#
- .NET
- Aplicação Console
- Visual Studio

---

# Aprendizados

Durante o desenvolvimento desta lista de exercícios, foram reforçados conceitos fundamentais como:

- Lógica de programação
- Organização de código
- Estruturação de menus interativos
- Manipulação de dados
- Resolução de problemas passo a passo

---

# Controle de Versão e Git

Durante o desenvolvimento deste projeto, também foi praticado:

- Inicialização de repositório Git
- Versionamento do código
- Criação de commits
- Envio do projeto para o GitHub
- Organização e documentação com README.md

Esse processo foi importante para compreender a importância do controle de versão no desenvolvimento de software, 
permitindo acompanhar a evolução do código e manter histórico das alterações realizadas.
